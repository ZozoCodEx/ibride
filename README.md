# iBride

Мобільна платформа та AI-асистент для наречених (Flutter).

## Вимоги

- Flutter 3.41+ (stable channel)
- Chrome (для веб-режиму)

## Запуск

```bash
# Встановити залежності
flutter pub get

# Запустити в Chrome
flutter run -d chrome
```

## Getting Started

FlutterFlow projects are built to run on the Flutter _stable_ release.
